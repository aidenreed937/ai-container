## Goal

创建一个 React + TypeScript + Vite 的示例项目，放在仓库目录：`demo/react-ts-vite`。

## Scope (Hard)

- 只允许修改/新增：`demo/react-ts-vite/`
- 禁止修改仓库其它任何文件（包括配置、README、脚本等）

## Idempotency

- 如果 `demo/react-ts-vite/package.json` 已存在：直接退出，不要重复创建/重装依赖

## Steps

1. 在 `demo/` 目录执行：
   - `npm create vite@latest react-ts-vite -- --template react-ts --no-interactive`
2. 在 `demo/react-ts-vite/` 目录执行：
   - `npm install`
   - `npm run build`

## Output

- 如果失败：输出失败原因与失败命令，并退出（不再继续其他步骤）
